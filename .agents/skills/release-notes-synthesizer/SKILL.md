# Release Notes Synthesizer

## Workflow

1. Review recent commits or PR descriptions.
2. Group changes by feature, fix, and docs.
3. Call out breaking changes and migrations.
4. Provide a concise summary for stakeholders.

## Output

- Produce a Markdown release note section.
