# Minimal Repro Builder

## Workflow

1. Identify the smallest scene or route to reproduce the issue.
2. Reuse existing assets instead of adding new ones.
3. Add concise logging and cleanup code.
4. Document how to run the repro.

## Output

- Provide the new file locations and reproduction steps.
