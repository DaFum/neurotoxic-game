# CI Hardener

## Workflow

1. Inspect existing CI workflows and required checks.
2. Add caching and parallelization where safe.
3. Ensure lint, test, and build are enforced.
4. Summarize changes and expected impact.

## Output

- Provide updated workflow changes and reasoning.
